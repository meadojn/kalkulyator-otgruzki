PWA (iPhone/Android) — Калькулятор Отгрузки

Как запустить на iPhone:
1) Залей папку на хостинг (лучше всего: GitHub Pages / Netlify / Cloudflare Pages).
2) Открой ссылку в Safari.
3) Нажми Поделиться -> На экран «Домой».

Важно: PWA работает только по HTTPS (GitHub Pages/Netlify дают HTTPS автоматически).
